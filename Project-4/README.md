# Project 4: Inferential Statistics and Regression

This project applies inferential statistical methods to test hypotheses and analyze relationships between variables using linear and logistic regression in R.

## Objectives
- Test hypotheses using inferential statistics
- Build and interpret linear regression and multiple linear regression models
- Build and interpret logistic regression models

## Methods Used
- Hypothesis testing 
- Linear regression
- Logistic regression
