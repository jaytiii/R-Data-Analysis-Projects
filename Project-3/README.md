# Project 3: Descriptive Statistics and Missing Value Imputation in R

This project applies descriptive statistical methods to summarize a dataset and addresses missing values using data imputation techniques.
The goal is to improve data quality and reliability before further analysis.

## Objectives
- Summarize database using descriptive statistics
- Identify missing values in the dataset
- Apply data imputation techniques to handle missing data
- Compare distributions before and after imputation
